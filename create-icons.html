📝 Instruções

1. Clique em "Gerar e Baixar Todos" para criar todos os ícones.

2. Os ícones serão baixados individualmente.

3. Crie a pasta: C:\Evidence\assets\icons\

4. Cole todos os ícones baixados lá.

5. O favicon.ico deve ser salvo na raiz: C:\Evidence\favicon.ico

6. Commit e push para o GitHub.
