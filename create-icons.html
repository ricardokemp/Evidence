📝 Instruções

1. Clique em "Baixar Todos os Ícones" para gerar todos os ícones.

2. Extraia os arquivos para: /assets/icons/

3. Os ícones serão gerados nos tamanhos necessários para o PWA.

4. O pacote inclui todos os ícones do manifest.json.

5. Os ícones são gerados com o símbolo 🔍 e gradiente roxo.

📄 Configuração do Manifest

{
  "icons": [
    {
      "src": "assets/icons/icon-72x72.png",
      "sizes": "72x72",
      "type": "image/png",
      "purpose": "any maskable"
    },
    ...
  ]
}
